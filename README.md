# Ecuacion parametrica

## Descripción

Este repositorio contiene la solución en Python de diferentes ejercicios relacionados con:

- Ecuaciones paramétricas
- Ecuaciones de la recta
- Ecuaciones cartesianas del plano
- Intersección entre recta y plano
- Aplicación práctica utilizando vectores y planos

Los ejercicios fueron desarrollados mostrando:

- Datos
- Procedimiento
- Resultado final

---

# Contenido del repositorio

## 1. resolucion_ejercicios.py

### Ecuaciones paramétricas y ecuaciones de la recta

Se desarrollan 5 ejercicios mostrando:

- Punto inicial
- Vector dirección
- Ecuaciones paramétricas
- Despeje del parámetro
- Ecuación simétrica de la recta

### Ecuaciones cartesianas del plano

Se resuelven dos ejercicios utilizando:

- Producto cruz
- Vector normal
- Ecuación cartesiana del plano

Además, cada ejercicio se desarrolla tomando como origen cada uno de los tres puntos dados.

### Intersección entre recta y plano

Se calcula:

- La ecuación paramétrica de la recta
- La ecuación del plano
- Sustitución
- Valor del parámetro
- Punto de intersección

---

## 2. aplicacion_practica.py

Este archivo contiene una aplicación práctica donde:

- Un dron se desplaza siguiendo una trayectoria recta
- Se calcula el punto donde atraviesa un plano de control

El programa muestra:

- Datos del problema
- Ecuaciones paramétricas
- Procedimiento matemático completo
- Sustitución
- Cálculo del parámetro
- Punto de intersección
- Verificación del resultado
